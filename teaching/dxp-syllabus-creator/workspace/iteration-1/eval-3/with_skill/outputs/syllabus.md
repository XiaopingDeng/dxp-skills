# 混凝土结构课程设计  课程教学大纲

## 一、课程设计的性质和任务

### 1. 课程性质

本课程是土木工程专业的一门专业必修实践课，属于课程设计类教学环节。以混凝土结构基本原理、建筑结构抗震设计等理论课程为基础，通过完成一个完整的混凝土结构工程设计任务，培养学生综合运用所学知识解决实际工程问题的能力。

### 2. 课程任务

本课程设计的主要任务是：在教师指导下，学生独立完成某多层钢筋混凝土框架结构房屋的楼盖设计、框架结构内力分析与设计、基础设计等主要环节，完成设计计算书和施工图绘制。通过本课程设计，学生应掌握混凝土结构设计的基本流程、设计方法和构造要求，建立正确的工程结构设计理念，培养严谨的科学态度和工程伦理意识。

### 3. 课程目标

**课程目标1（知识目标）：** 掌握混凝土结构设计的基本原理、设计方法和构造要求，熟悉现行《混凝土结构设计规范》（GB 50010）、《建筑结构荷载规范》（GB 50009）等相关规范的主要规定，能够正确应用规范进行结构设计计算。

**课程目标2（能力目标）：** 能够独立完成多层钢筋混凝土框架结构的结构布置、荷载计算、内力分析、截面设计及构造设计，具备使用结构设计软件（如PKPM/YJK）进行辅助设计的能力，能够正确绘制结构施工图。

**课程目标3（素养目标）：** 培养严谨求实的科学态度、精益求精的工匠精神，树立工程安全意识和社会责任感，理解结构工程设计中的伦理问题，在设计中体现可持续发展理念。

**课程目标4（创新目标）：** 能够在满足规范要求的前提下，对结构方案进行合理优化，具备初步的工程创新意识和能力。

### 重点与难点

**重点：**
1. 结构布置方案的合理确定与荷载计算
2. 现浇肋梁楼盖的设计方法与构造要求
3. 框架结构的内力分析方法（分层法、D值法）
4. 框架梁、柱截面承载力设计方法

**难点：**
1. 框架结构在水平荷载作用下的内力分析与侧移计算
2. 梁柱节点的构造设计与配筋处理
3. 不同荷载组合下的最不利内力确定
4. 结构方案的优化与工程判断

## 三、各教学环节要求

### 1. 课前预习

学生应在每次设计课前预习相关的规范条文、教材章节和设计手册内容，了解本次设计任务的技术要求和规范依据。

### 2. 课堂讲授与指导

教师讲授设计方法、规范应用要点和常见问题分析，全程指导学生完成设计任务，采用启发式教学引导学生独立思考。鼓励学生利用结构设计软件（PKPM/YJK）进行方案验证和优化。

### 3. 自主设计与计算

学生应在规定时间内独立完成设计计算书的编制，包括：
- 结构布置方案说明
- 荷载计算书（含荷载标准值、设计值、荷载组合）
- 楼盖设计计算书（板、次梁、主梁）
- 框架内力分析计算书
- 框架截面设计与配筋计算书
- 基础设计计算书

### 4. 施工图绘制

学生应完成以下施工图绘制（可采用CAD或BIM软件）：
- 标准层结构布置图
- 楼盖配筋图（板配筋图、梁配筋图）
- 框架梁、柱配筋详图
- 基础平面布置图及配筋详图

### 5. 成果答辩

学生提交完整的设计成果（设计计算书+施工图+设计总结报告），参加课程设计答辩。答辩内容包括：设计思路阐述、设计方法介绍、主要技术参数的确定依据、设计中的难点及解决方法、自我评价与改进方向。

## 六、课程教学改革摘要

### 1. 立德树人

深度挖掘混凝土结构课程设计中的思政元素，将工程伦理教育融入设计的每个环节。在规范应用教学中强调"规范是安全的底线"，培养学生对人民生命财产安全高度负责的态度；在结构方案设计中引入"强柱弱梁、强剪弱弯"的设计理念，类比社会生活中的担当精神；结合我国超高层建筑和大型基础设施建设的辉煌成就，激发学生的专业自豪感和家国情怀。

### 2. 教学思想

全面落实"以学生为中心、以产出为导向、持续改进"的OBE教育理念。以"能够独立完成混凝土结构设计任务"为最终产出目标，反向设计教学内容体系和考核标准。在教学过程中，引导学生主动思考、自主探究，由"教师讲授"向"教师引导+学生自主设计"转变，提升学生的工程实践能力和创新能力。

### 3. 教学方法

采用"理论+实践+创新"三维教学模式：
- **理论维度**：以规范讲解和案例教学为基础，夯实学生的结构设计理论知识
- **实践维度**：以真实工程任务为载体，学生在完成设计任务的过程中掌握设计方法和技能
- **创新维度**：鼓励学生在方案设计中尝试多种方案比选和优化，培养工程创新思维

### 4. 教学手段

- **智慧教学平台**：利用超星学习通/雨课堂等平台发布任务、答疑讨论、过程管理
- **AI辅助工具**：引导学生合理使用结构设计AI辅助工具进行规范查询、计算复核
- **软件仿真教学**：运用PKPM/YJK结构设计软件进行结构建模与分析，实现设计方案的对比验证
- **数据驱动改进**：通过过程评价数据追踪学生学习轨迹，及时发现薄弱环节，实施精准辅导
